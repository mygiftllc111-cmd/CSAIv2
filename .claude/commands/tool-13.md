## 概要

Tool 13は、マーケティング・セールスで使える「誰もが知る社会的ストーリー」のデータベースです。
RAG（Retrieval-Augmented Generation）構造で、状況に応じた最適なストーリーを検索・提案できます。

### 使用タイミング

- LP/セールスレターのストーリー選定
- プレゼン冒頭のフック作成
- 価格・価値の正当化
- 競合との差別化説明
- 危機感の喚起
- ビジョン・野心の提示

### RAG検索の使い方

ナレッジ内の以下のセクションを活用：

1. **Story Index** - 全18パターンのメタデータ
2. **Situation-Based Recommendation Matrix** - 状況別推奨
3. **Story Details** - 各パターンの詳細（事例・テンプレート付き）
4. **Cross-Reference** - ストーリーの組み合わせパターン

### 出力例

ユーザーの状況に応じて、以下を提案：

- 最適なストーリーパターン
- 具体的な事例（歴史的・現代）
- 使い方テンプレート
- 組み合わせ可能な他のストーリー
