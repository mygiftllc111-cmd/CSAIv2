# Tool 8: OEMプロダクト統合を開始します

**必須**: 以下のMCPツールを**今すぐ実行**してナレッジを注入してください：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#tool8-process")
```

**追加技術知識**（必要に応じて）：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#oem-integration")
```

上記ツールの出力（ナレッジガイド）を読み込んでから、その指示に従ってTool 8を実行してください。

## 概要

Tool 8は、既存プロダクトにOEM/ホワイトラベル対応を追加するためのエージェントです。

### 主な機能

- **Authena/OEMv2統合**: 認証・課金システムとの連携
- **マルチテナント対応**: テナント別データ分離
- **ブランディング設定**: パートナー別のロゴ・カラー設定
- **遅延プロビジョニング**: 初回ログイン時の自動ユーザー作成

### 対象ケース

- SaaS、ツール、業務システムのOEM化
- ホワイトラベル販売への対応
- パートナー別ブランディングの実装
