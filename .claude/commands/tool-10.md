# Tool 10: プロダクトテストを開始します

**必須**: 以下のMCPツールを**今すぐ実行**してナレッジを注入してください：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#tool10-process")
```

**追加技術知識**（必要に応じて）：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#tool10-tech")
```

上記ツールの出力（ナレッジガイド）を読み込んでから、その指示に従ってTool 10を実行してください。

## 概要

Tool 10は、追加したプロダクト単体の全フローを高速テストするエージェントです。

### 特徴

- **高頻度・高効率**: プロダクト追加のたびに実行
- **並列デバッグ**: デバッグ知識蓄積エージェントを同時起動
- **3ロールテスト**: スーパーアドミン・アドミン・エンドユーザー
- **即時修正**: 問題発見→修正→再テストのサイクル

### 使用タイミング

- 新規プロダクト追加後（毎回）
- プロダクト設定変更後
- プラン・価格変更後
- ブランディング設定変更後

### テスト範囲

1. **プロダクト設定確認**（プラン・価格・リソース）
2. **3ロール別フロー**
3. **ブランディング表示**
4. **認証・ログイン**
5. **プラン制限動作**
6. **エラーハンドリング**

### 並列エージェント

テスト中、以下のエージェントを並列起動して効率化：

- **デバッグ知識蓄積エージェント**: エラーパターンと解決策を記録
