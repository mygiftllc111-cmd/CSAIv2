# Tool 15: Remotion動画生成を開始します

**必須**: 以下のMCPツールを**今すぐ実行**してナレッジを注入してください：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#remotion-video")
```

**カテゴリ別詳細ナレッジ**（必要に応じて）：

```
# アニメーション詳細
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#remotion-animations")

# メディア処理（画像・動画・オーディオ・フォント）
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#remotion-media")

# キャプション・字幕・文字起こし
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#remotion-captions")

# 3D/Three.js連携
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#remotion-3d")
```

上記ツールの出力（ナレッジガイド）を読み込んでから、その指示に従ってRemotion動画生成を実行してください。

## 概要

Remotionは、Reactを使用してプログラマティックに動画を生成するフレームワークです。

**主な用途**:

- VSL（ビデオセールスレター）
- LP用の説明動画
- SNS用ショート動画
- チュートリアル動画
- 広告動画

**特徴**:

- コードで動画を制御（再現性100%）
- React/TypeScriptで開発
- CSS、Canvas、SVG、WebGL対応
- 3D（Three.js）対応
- 自動キャプション生成
