# Phase 3: フロントエンド基盤を開始します

**必須**: 以下のMCPツールを**今すぐ実行**してナレッジを注入してください：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#phase3-process")
```

**追加技術知識**（必要に応じて）：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#phase3-tech")
```

上記ツールの出力にはライセンス状態に応じた権限情報が含まれています。
その指示に従ってPhase 3を実行してください。

完了後は `gambit complete-phase 3` を実行して進捗を記録します。
