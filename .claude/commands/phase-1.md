## ステップ1: ナレッジ注入（必須）

以下のMCPツールを**今すぐ実行**してください：

```
mcp__gambit-ai-gen-knowledge__gambit_get_knowledge(keyword: "#phase1-requirements")
```

**警告**: 上記ツールが実行できない場合は、MCPサーバーが接続されていません。以下を確認してください：

- Claude Codeを再起動してください
- ~/.claude/settings.json にMCPサーバー設定があるか確認
