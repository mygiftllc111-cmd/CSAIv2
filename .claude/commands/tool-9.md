**必須**: 以下のMCPツールを**今すぐ実行**してナレッジを注入してください：


**追加技術知識**（必要に応じて）：

```

```

上記ツールの出力（ナレッジガイド）を読み込んでから、その指示に従ってTool 9を実行してください。

## 概要

Tool 9は、OEMシステム全体の統合テストを実行し、**販売可能レベル**を検証するエージェントです。

### 特徴

- **低頻度・高品質**: 追加実装時や大きな変更後に実行
- **詳細ドキュメント**: テスト結果を細かく記録
- **コミット記録**: 各修正をGitコミットで追跡
- **3ロールテスト**: スーパーアドミン・アドミン・エンドユーザー

### 使用タイミング

- 新規プロダクトのOEM統合完了後
- 大規模な機能追加・改修後
- 本番リリース前の最終確認
- 定期的な品質監査

### テスト範囲

1. **OEMProduct登録フロー**
2. **パートナー招待・登録フロー**
3. **エンドユーザー登録・認証フロー**
4. **ブランディング設定・表示**
5. **権限・プラン制限**
6. **エラーケース・エッジケース**
